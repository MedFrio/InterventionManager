# InterventionManager

[![C#](https://img.shields.io/badge/C%23-.NET-512BD4?logo=dotnet&logoColor=white)](https://dotnet.microsoft.com/)

Solution de gestion d'interventions conçue pour organiser, suivre et fiabiliser les opérations terrain.

## Fonctionnalités

- Gestion des interventions et de leur statut
- Structuration des flux métier techniques
- Base applicative maintenable pour extensions
- Orientation usage professionnel

## Stack technique

- C#
- .NET
- Solution Visual Studio

## Prérequis

- .NET SDK compatible
- Visual Studio ou CLI dotnet

## Démarrage rapide

```bash
dotnet restore
dotnet build
```

## Commandes utiles

```bash
dotnet build
dotnet test
```
